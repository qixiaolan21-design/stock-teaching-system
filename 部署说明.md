# 部署说明

## 问题：Render 看不到仓库

### 解决方案：

#### 方法1：重新授权 GitHub
1. 打开 https://dashboard.render.com/settings
2. 点击 **GitHub** 旁边的设置
3. 点击 **Configure**
4. 确保授予了 `stock-teaching-system` 仓库的访问权限

#### 方法2：手动连接
1. 打开 https://dashboard.render.com/new/web-service
2. 选择 **Public Git repository**
3. 输入仓库地址：
   ```
   https://github.com/qixiaolan21-design/stock-teaching-system
   ```

#### 方法3：使用 Render Blueprint
1. 打开 https://dashboard.render.com/blueprints
2. 点击 **New Blueprint Instance**
3. 选择 `stock-teaching-system` 仓库
4. Render 会自动读取 `render.yaml` 配置

---

## 部署配置

如果手动创建，请使用以下配置：

| 配置项 | 值 |
|--------|-----|
| Name | stock-teaching-system |
| Runtime | Node |
| Build Command | `npm install` |
| Start Command | `npm start` |
