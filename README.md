# 📈 股票案例印证知识库

股票案例印证知识库系统，用于记录、管理和检索YouTube直播中的股票分析案例。

## 功能特点

- 📚 **案例记录** - 记录股票分析案例（支撑/压力/止损/目标）
- 📑 **PPT库** - 上传PPT文件，自动提取所有页面为图片
- 🔗 **案例关联PPT** - 案例可以关联到PPT的具体页面
- 🔍 **智能搜索** - 搜索案例和PPT，展示匹配的PPT内容
- 💬 **学员反馈** - 记录学员获利反馈
- 🤖 **AI提取** - 自动从文本提取股票/价格/方法
- 🏷️ **标签系统** - 按标签快速筛选
- 📊 **数据统计** - 案例趋势图表

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 打开浏览器

访问 http://localhost:3000

## 使用说明

### 上传PPT到PPT库

1. 点击 **"📤 上传PPT"** 按钮
2. 选择 `.pptx` 文件
3. 系统自动提取PPT中的所有页面为图片
4. 填写标题、日期、类型等信息
5. 点击保存到PPT库

### 添加案例并关联PPT

1. 点击 **"➕ 案例"** 按钮
2. 填写案例信息（股票名称、代码、日期、方法等）
3. 在"关联PPT"下拉框中选择一个PPT
4. 选择该案例对应的PPT页面
5. 保存案例

### 查看案例详情

- 点击案例卡片查看详情
- 如果案例关联了PPT，会显示对应的PPT页面图片
- 点击图片可以查看大图

### 搜索功能

- 在搜索框输入关键词（股票名称、代码、方法、标签等）
- 系统会同时搜索：
  - 案例库中的匹配案例
  - PPT库中的匹配PPT
- 搜索结果会展示匹配的PPT页面预览

### 查看PPT库

- 点击导航栏 **"📑 PPT库"**
- 查看所有上传的PPT
- 点击PPT卡片查看所有页面

## 数据备份

在"数据"页面可以导出/导入 JSON 数据，建议定期备份。

## 注意事项

- 只支持 `.pptx` 格式的PPT文件
- 上传的PPT文件和图片会保存在服务器上
- 数据文件保存在 `data/` 目录下
- 上传的文件保存在 `uploads/` 目录下
