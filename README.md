# 股票教学案例管理系统

专为 YouTube 股票教学主播设计的运营工具

## 五大核心功能

### 1. 案例记录
- 上传 Word/PDF/TXT 文档
- 自动提取股票代码、名称
- 自动识别：压力位、支撑位、风险点、机会点
- 按时间整理所有历史案例

### 2. 印证案例库
- 上传 PPT 课件
- 逐页解析，提取每页股票信息
- 标签化：筹码、机构DC、密码、趋势、量能、K线等
- 快速搜索：输入"筹码"筛选所有筹码相关案例
- 精准定位到 PPT 具体页面

### 3. 会员反馈
- 整理会员获利截图
- 标签化：赢家K线、筹码、飘带、机构DC等方法
- 记录获利金额

### 4. 图表展示
- 纳指 K 线走势图
- 在图表上标记教学日期
- 直观展示指数低点与教学内容的关系
- 验证讲课时机的准确性

### 5. 股票验证（高级）
- 批量验证股票涨跌
- 对比讲课时的预测与实际走势

## 技术栈
- Node.js + Express
- 前端：原生 JS + Tailwind CSS + ECharts
- PPT 解析：JSZip

## 部署
支持 Render、Vercel 等平台
